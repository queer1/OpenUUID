What is OpenUUID?

OpenUUID is an open source universally unique identifier (UUID) generator for OS X.

Check out project's GitHub repo here.

Screenshot

Description

The source code served as an example of how to get certain types of UUID on Mac OS X, it is, by no means, a commercial grade product. It might contain errors or flaws, and it was created for demonstration purpose only.

More information can be find on Wikipedia.

Usage

Press the buttons to generate the UUID.
